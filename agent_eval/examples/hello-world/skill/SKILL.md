## When to use this

Use when asked to create a greeting or hello world file.

## What to do

Create a file called `hello.txt` containing a friendly greeting message.

The greeting should:
- Start with "Hello"
- Be friendly and welcoming
- Be 1-3 sentences

## Example output

```
Hello! Welcome to the world of coding. Have a great day!
```
