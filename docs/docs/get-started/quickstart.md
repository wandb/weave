# Quickstart

You will learn:
- How to using tracing to debug inputs & outputs of each function
- How to log models & datasets and see when they're used
- How to use the Weave UI to gain insights

## Installation

`pip install weave`

## Start logging

```
import weave

weave.init('my-first-project')
```
